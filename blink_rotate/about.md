# Rotate Blink

## Parts Used

- Arduino Uno
- Breadboard
- 560 Ohm resistor
- Green LED
- A whole bunch of male-male and female-female cables
- Rotary encoder

## What this does

Each time the rotary encoder is turned one notch, the LED will blink once.
